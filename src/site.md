# Work

Case studies shown on the home page, in display order.

To add a new project:
  1. Create a .md file in src/content/ with `type: case-study` in the front matter.
  2. Add its slug to this list in the position you want it to appear.
  3. Run `npm run build`.

If a case study file exists in src/content/ but is NOT listed here,
it will still be built and linked — it just won't appear in the home page grid
until you add it to this list.
